# Quantium Data Analytics Simulation

This repository contains my work for the Quantium Data Analytics Virtual Internship. The project focuses on using data-driven insights to help a Category Manager optimize the chips category.

## Project Highlights
* [cite_start]**Data Cleaning:** Handled outliers (bulk purchases) and filtered non-chip products to ensure data integrity. [cite: 97, 107]
* [cite_start]**Customer Segmentation:** Identified that **Mainstream Young Singles/Couples** and **Budget Older Families** are the primary sales drivers. [cite: 216]
* [cite_start]**Uplift Testing:** Selected control stores using Pearson Correlation and Magnitude Distance to evaluate the impact of new store layouts. [cite: 44, 189]
* [cite_start]**Commercial Insights:** Delivered a final recommendation to the client using the Pyramid Principle. [cite: 245]
